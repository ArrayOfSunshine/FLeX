Settings

Game version

This changes the page theme and the version used when loading encounter data. Existing saved data is kept.

Move index

Used by the Dex move filter. It is built once and cached in this browser.

Ability cache

Builds and caches Gen 3 ability data for all 386 Pokémon so Dex ability sections and filters do not need to fetch each Pokémon one-by-one later.

Locations cache

Builds and refreshes exact wild encounter data used by the Locations tab and each Pokémon's Dex location section.

Progress card testing

Temporarily shows the celebration backgrounds on supported Progress cards without changing any saved Pokédex or Living Dex values.

Backup / restore all data

Exports or restores all saved data for this tracker, including Dex, My Mon's, Teams, Settings, move indexes, and encounter caches.

Reset saved data

Progress is saved in this browser using localStorage. Use Export/Import if moving devices or browsers. Move data comes from PokéAPI and is cached locally after loading.
